Definition 3.1.6.

Let \(X\) be a set and \(\Sigma\) a \(\sigma\)-algebra on \(X\text{.}\) A set function \(\mu:\Sigma \to [0,\infty]\) is a measure if

  • For all \(E \in \Sigma\text{,}\) \(\mu(E) \geq 0\text{.}\)

  • \(\mu(\emptyset) = 0\text{.}\)

  • For all countable collections \(\{E_j\} \subset \Sigma\) of pairwise disjoint sets,

    \begin{equation*} \mu(\bigcup_j E_j) = \sum_j \mu(E_j). \end{equation*}
