We'll proceed by cases on the assumption that \(\th(x) \nsim x\text{.}\) First, assume that there exists \(\la > 1\) so that \(\th(x) > \la x\) for sufficiently large \(x\text{.}\) Given such an \(x\text{,}\) as \(\th\) is an increasing function,

\begin{equation*} \int_x^{\la x} \frac{\th(t) - t}{t^2} \, dt \geq \int_x^{\la x} \frac{\la x - t}{t^2}\, dt = \int_1^\la \frac{\la - s}{s^2} \, ds > 0, \end{equation*}

where this constant does not depend on \(x\text{.}\) However, the convergence in Step V means that for all \(\eps > 0\text{,}\) there exists some \(M\) for which \(M_1, M_2 \geq M\) we have

\begin{equation*} \int_{M_1}^{M_2} \frac{\th(x) - x}{x^2} \, dx \lt \eps, \end{equation*}

and so such a \(\la\) cannot exist.

Likewise, if there exists \(\la \lt 1\) with \(\th(x) \leq \la x\) for sufficiently large \(x\text{,}\) then for \(t \leq x,\)

\begin{equation*} \th(t) \leq \th(x) \leq \la x, \end{equation*}

and so

\begin{equation*} \int_{\la x}^{x} \frac{\th(t) - t}{t^2} \, dt \geq \int_{\la x}^{ x} \frac{\la x - t}{t^2}\, dt = \int_\la^1 \frac{\la - s}{s^2} \, ds \lt 0, \end{equation*}

which leads to the same contradiction. Thus,

\begin{equation*} \limsup_{x \to \infty} \frac{\th(x)}{x} \leq 1 \leq \liminf_{x\to \infty} \frac{\th(x)}{x}, \end{equation*}

which means that \(\th(x) \sim x\text{.}\)
