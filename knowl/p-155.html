A measure is a function that assigns a numerical value to a set. The idea comes from an effort to generalize the notion of the length of an interval. A brief overview of the setup follows. We have a set \(X\) (for example, \(X=\R\)), and we want to define a function \(\mu: A \subset X \to [0,\infty]\) (note the inclusion of \(\infty\)).
