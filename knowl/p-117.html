First, we'll translate the boundary conditions to the upper half plane using the usual conformal map
\begin{equation*} z \mapsto w = C(z) = i \frac{1+z}{1-z}. \end{equation*}
In the halfplane, we can find a function \(\phi(w)\) fitting the standand half plane solution by
\begin{equation*} \phi = 1 + \frac{0 - 1}{\pi } \theta_1 + \frac{1 - 0}{\pi} \theta_2 \end{equation*}
which has complex potential
\begin{equation*} \Phi = 1 + \frac{0 - 1}{\pi i} \log(w+1) + \frac{1 - 0}{\pi i} \log(w -1). \end{equation*}
