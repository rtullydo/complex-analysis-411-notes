Proof

The argument is combinatorial-esque. For \(n \in \mathbb{N}\text{,}\) we have

\begin{align*} 2^{2n} = (1+1)^{2n} \amp= \binom{2n}{0} + \ldots \binom{2n}{2n} \\ \amp\geq \binom{2n}{n}\\ \amp\geq \prod_{n \leq p \leq 2n} p\\ \amp= \exp(\th(2n) - \th(n)). \end{align*}

This implies that

\begin{equation*} \th(x) - \th(\frac{x}{2}) \leq C x \end{equation*}

for any \(C > \log 2\) for all \(x > x_0\) for some \(x_0\) whose choice depends on \(C\text{.}\) Now, given some large value of \(x\text{,}\) consider the sequence

\begin{equation*} x, \frac{x}{2}, \frac{x}{4}, \ldots, \frac{x}{2^r} \geq x_0 \geq \frac{x}{2^{r+1}}.\text{.} \end{equation*}

The corresponding inequalities are

\begin{align*} \th(x) - \th(\frac{x}{2}) \amp\leq Cx\\ \th(\frac{x}{2}) - \th(\frac{x}{4}) \amp\leq C\frac{x}{2}\\ \vdots\\ \th(\frac{x}{2^{r}})- \th(\frac{x}{2^r+1}) \amp\leq C\frac{x}{2^{r}} \end{align*}

Adding these results in the inequality

\begin{equation*} \th(x) - \th(\frac{x}{2^{r+1}}) \leq (1 + \frac{1}{2} + \ldots + \frac{1}{2^{r}})Cx \leq 2Cx, \end{equation*}

and so, noting that \(\th(\frac{x}{2^{r+1}}) \leq \th(x_0)\text{,}\) we get

\begin{equation*} \th(x) \leq 2Cx + \th(x_0), \end{equation*}

or in order notation

\begin{equation*} \th(x) = \mathrm{O}(x) \text{ as } x\to \infty \end{equation*}

as \(\th(x_0) = \mathrm{O}(1)\text{.}\)
