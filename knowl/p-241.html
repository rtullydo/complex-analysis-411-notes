Likewise, if there exists \(\la \lt 1\) with \(\th(x) \leq \la x\) for sufficiently large \(x\text{,}\) then for \(t \leq x,\)

\begin{equation*} \th(t) \leq \th(x) \leq \la x, \end{equation*}

and so

\begin{equation*} \int_{\la x}^{x} \frac{\th(t) - t}{t^2} \, dt \geq \int_{\la x}^{ x} \frac{\la x - t}{t^2}\, dt = \int_\la^1 \frac{\la - s}{s^2} \, ds \lt 0, \end{equation*}

which leads to the same contradiction. Thus,

\begin{equation*} \limsup_{x \to \infty} \frac{\th(x)}{x} \leq 1 \leq \liminf_{x\to \infty} \frac{\th(x)}{x}, \end{equation*}

which means that \(\th(x) \sim x\text{.}\)
