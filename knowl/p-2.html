By way of example, consider the function \(f(z) = z^2\) at the point \(z_0 = 1 + 1i = \sqrt{2}e^{i\pi/4}\text{.}\) The horizontal line through \(z_0\) can be parametrized by \(\gamma(t) = (1 + t) + 1i\) which passes through \(z_0\) at \(t = 1\text{.}\) The tangent vector at \(t = 1\) is \(\gamma\dd(0) = 1\text{.}\) Now look at the image. Composing, we get

\begin{equation*} f(\gamma(t)) = (t + 1i)^2 = (t^2 - 1) + 2ti. \end{equation*}

Then calculating the tangent vector to the image gives

\begin{equation*} f\dd(\gamma(t))|_{t = 1} = (2t + 2i)|_{t =1} = 2 + 2i. \end{equation*}

That is, the tangent vector has increased in length by a factor of \(2\sqrt{2}\) and undergone a rotation of \(\pi/4\) radians. If we check the vertical line through \(z_0\text{,}\) which can be parametrized by \(\gamma_2(t) = 1 + 1ti\) where \(\gamma_2(1) = z_0\text{,}\) we find that the tangent vector \(\gamma_2(1) = i\) has image \(f(\gamma_2(1)) = -2 + 2i\text{,}\) which again represents an increase in magnitude of \(2\sqrt{2}\) and a rotation of \(\pi/4\) radians.
